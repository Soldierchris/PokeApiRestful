La siguiente APIResful conserva los principios de arquitectura del MVC, organizado dentro de la carpeta src

API_POKEMON/
│
├── src/
│   ├── models/
│   │   ├── pokemon.model.js
│   │   └── user.model.js
│   │
│   ├── controllers/
│   │   ├── auth.controller.js
│   │   └── pokemon.controller.js
│   │   
│   │
│   ├── routes/
│   │   ├── auth.routes.js
│   │   ├── pokenon.routes.js
│   │   └── index.js
│   │

│   │   └── validationPokemon.js
│   │
│   ├── services/
       └── pokemon.service.js


La APi Utiliza las siguientes librerias:

dotenv@16.4.7
